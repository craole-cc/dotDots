# Nix Style Guide

This directory splits the style guidance by concern.

## Files

- `shared.md` — rules that apply across the Nix stack
- `libraries.md` — `Libraries/nix` patterns and examples
- `modules.md` — `Modules/nix` patterns and examples
- `api.md` — `API/nix` data-shape conventions

## Recommended reading order

1. `shared.md`
2. `libraries.md`
3. `modules.md`
4. `api.md`

If you only need the top-level index, keep using `Documentation/nix/styleguide.md` as the compatibility entrypoint.
